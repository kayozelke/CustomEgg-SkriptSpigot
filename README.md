# CustomEgg-SkriptSpigot
Skript for Spigot Skript plugin that allows to execute any command when right clicking by player.

Example usage scenario: Spawning a mob with specific NBT data. This custom mob egg cannot be used in the mob spawner as default ones do. 

Requirements:
- Skript version 2.13.1
- SkBee version 3.13.2
- Spigot 1.21.10 (tested on Purpur version 1.21.10-2528-HEAD@95a466c)

Usage and permissions - read the script files, the script language is similar to english.
